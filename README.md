## 指令
- `$help [插件]` 指令列表
- `$init` 初始化 機器人
- `$plugin install <插件>` 安裝 插件
- `$plugin uninstall <插件>` 卸載 插件
- `$plugin info [插件]` 插件 資訊
- `$permission <用戶> <等級>` 設定用戶權限等級
- `$permission <用戶>` 查詢用戶權限等級
- < > 表示該參數為必填  [ ] 表示該參數為選填
